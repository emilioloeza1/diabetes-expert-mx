¿Cómo funciona este sistema?

El sistema utiliza un motor de inferencia basado en reglas if-then. Cada síntoma tiene un peso diagnóstico (1–3). El motor suma estos pesos y aplica modificadores si hay factores de riesgo presentes.

Adicionalmente, evalúa reglas especiales: si están presentes los 3 síntomas cardinales (triada: poliuria + polidipsia + polifagia), el nivel mínimo se eleva a Moderado. Si existe combinación de síntomas de progresión con cardinales, se añaden puntos extra.

Limitaciones: Este sistema NO reemplaza una consulta médica. No toma en cuenta análisis clínicos (glucosa, HbA1c), historial completo, ni medicamentos. Es una herramienta de orientación inicial.
